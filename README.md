README — sistem-kelompok-akademik

Academic Group Management System

A web-based system to manage academic group assignments.

Features


Login / logout with role-based access (admin & student)
Admin dashboard with live statistics (students, groups, courses, assignments)
Full CRUD for:

Student data
Group data
Course data
Group assignments





Database

6 tables: mahasiswa, kelompok, mata_kuliah, tugas_kelompok, anggota_kelompok, user

Tech


PHP
MySQL
XAMPP
HTML / CSS


How to Run


Install XAMPP
Start Apache and MySQL
Import database via phpMyAdmin
Open localhost/sistem-kelompok-akademik in browser


Status

Core features fully functional. Planned: data validation, success notifications, UI improvements.
